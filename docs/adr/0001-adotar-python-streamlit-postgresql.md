# Adotar Python, Streamlit e PostgreSQL

Status: accepted

O LabVida sera implementado como um prototipo academico usando Python, Streamlit, PostgreSQL, Docker, SQLAlchemy, Alembic e Pydantic. A decisao substitui a stack de referencia citada nas entregas anteriores, pois a stack atual foi definida como obrigatoria pelo professor para a etapa de implementacao.
